Watchlists

O botão de rádio define a lista ativa (é nela que a estrela adiciona). Edite o nome no campo. As listas ficam salvas no seu navegador.

Colunas da lista

Escolha quais indicadores aparecem na tabela. Ativo e Score são fixos. Colunas marcadas com fund só têm dado para ações e units no plano free; as com enr exigem enriquecer o ativo.

Composição do Score

Ajuste o peso de cada grupo (0 = ignora, 1 = padrão, 2 = dobro). O score é normalizado de 0 a 100 conforme os pesos. Vale para ações e units; fundos usam só sinais técnicos.
